Catalyst

Extending QuakeStudies for a Learning Legacy

Presented by Dr Chris Thomson (UC) & Jonathan Hunt (Catalyst)

 

 

What's it all about?

  • A project to capture and disseminate lessons learned from SCIRT's rebuild of Christchurch's "horizontal infrastructure"

    • Not so horizontal...

    SCIRT's remit includes repairing wastewater, stormwater and freshwater infrastructure, roads, bridges and supporting structures for these (eg retaining walls).

    Who's involved?

    • SCIRT - documenting innovative work
    • QuakeStudies - preservation & flexible architecture
    • QuakeCentre - project management & key stakeholders
    • Resilient Organisations - external research group
    • Catalyst IT - found solutions & wrote the code

    Learning Legacy concept

    • Capture important information before it is lost
    • Develop a digital archiving programme for rebuild agencies
    • CEISMIC Learning Legacy Scholarships
    • An opportunity to open-source and develop the platform for future

    QuakeStudies

    • quakestudies.canterbury.ac.nz
    • More than 88,000 items public items contributed to CEISMIC:
      • ~41,000 newspaper pages
      • ~39,000 images
      • ~500 videos
      • ~120 audio files
      • Lots of other content: graphics, composite images, research presentations, blogs, etc.

    QS, LL Architecture

    Drupal Remote Entities

    • Fedora Commons REST API -> wrapper (Tuque from Islandora) -> Clients module
    • Declare to Drupal as remote entity: fedoraentity.module, bundles: Collection, Object, Part, License
    • In QuakeStudies: top collection/735 #
    • match: Communications and Community (collection:741 # -> /theme/communications-and-community)
    • Web Service Clients: config
    • local Solr
    • Some fields streamed from QuakeStudies: e.g. videos

    Lessons

    • Multi-stakeholder consortiums are hard work
    • Re-use is possible: of architecture, code
    • Open source communities delivering increasingly powerful components
    • Big problems can equal big opportunities to experiment and learn
